HTML

HTML stands for Hypertext Markup Language, and it is the standard markup language used to create web pages. HTML is responsible for structuring the content of a web page, including headings, paragraphs, images, and other elements.

CSS

CSS stands for Cascading Style Sheets, and it is used to add style and layout to web pages. CSS is responsible for defining the visual appearance of a web page, including font sizes, colors, and spacing. It can control the layout of multiple web pages all at once.

Python

Python is a high-level programming language that is widely used for a variety of tasks, including web development, data analysis, machine learning, and automation. Python is known for its readability, ease of use, and vast library of modules.

C Programming

C is a high-level programming language that is widely used for system programming, operating systems, and embedded systems. C is known for its efficiency, speed, and low-level access to hardware. C is very fast, compared to other programming languages.
